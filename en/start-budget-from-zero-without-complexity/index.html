Start a Budget from Zero Without Complexity

Launch your first budget in 15 minutes using four buckets, then use daily tracking and a weekly review to keep it realistic.

Open reports (web) Create your budget (app)

15-minute setup

  1. Use a safe income number (lowest 3-month average).
  2. Split into four buckets: essentials, wants, debt/savings, buffer (5–10%).
  3. Set weekly caps for wants; keep categories broad.
  4. Add one rule: any non-essential over $25 waits 24h.

Daily tracking (2 minutes)

Log right after purchase; tag essentials vs wants; note any spend >$25 for weekly review.

Weekly review (15–20 minutes)

  1. Totals: spend vs plan, buffer remaining.
  2. Top 3 categories over plan.
  3. One corrective move for next week (cap, delay, swap).
  4. Send any surplus to debt/emergency fund.

Starter example

  • Income: $2,400 | Essentials $1,250 | Buffer $180 | Wants $520 | Debt/Savings $450
  • Rule: wants over $25 wait 24h; dining max $80/week.

FAQ

Do I need detailed categories? No—broad buckets keep it simple.

What if income is variable? Recalculate weekly using a safe average and keep a larger buffer (10%).

When to adjust caps? During the weekly review; never in the moment of spending.
