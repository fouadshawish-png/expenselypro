App FAQ

What is the difference between tracking here and using a spreadsheet?

The app records time, account, and category in one flow, then updates balances and budgets immediately. A spreadsheet usually stays manual and requires repeated entry with no built-in warnings.

Do I need a constant internet connection?

No. Entries are stored locally first, then synced when the connection returns. You can track for days offline and sync later.

Can I use the app without creating an account?

Yes. You can start with anonymous access, then connect Google later if you want cloud recovery on another device.

What happens to my data if I delete the app?

If you use Google sign-in, your synced data can be restored after reinstalling. Anonymous local data is removed unless you kept a backup.

How does backup work?

You can export an encrypted backup file and import it later. Backup is manual and never runs without a clear action from you.
