Authority Structure

Goal

Build a coherent topical authority layer around expenses and budgeting using a Hub + Cluster architecture.

1) Architecture

  • Core hubs:
    • /إدارة-المصاريف/
    • /الميزانية/
    • /تتبع-المصاريف/
    • /الديون/
    • /الادخار/
  • Master guides:
    • /دليل-إدارة-المصاريف-والميزانية/
    • /financial-guide/
    • /money-management-guide/

2) Internal Link Rule (Mandatory)

Every article must include:

  1. A link to a primary pillar or hub article.
  2. A link to /download.html.
  3. A link to a comprehensive guide.
  4. A link to a supporting sub-article.

Minimum: 5 useful internal links.

3) Standard Article Template

Fixed structure

  1. Strong psychological hook (2–3 lines).
  2. Clear problem.
  3. Concrete numeric example.
  4. Practical framework (steps or model).
  5. Comparison (when relevant).
  6. FAQ.
  7. Dual CTA:
    • Reading CTA: link to a guide or hub article.
    • Action CTA: download link or deep link.

Reusable CTAs

  • Read: /دليل-إدارة-المصاريف-والميزانية/
  • Download: /download.html
  • Deep links:
    • /?route=reports
    • /?route=budgets
    • /?route=debt_ledger
    • /?route=goals

4) Update Cycle (every 60 days)

  1. Refresh the top 5 articles.
  2. Add one new numeric example to each refreshed article.
  3. Add or improve one FAQ per article.
  4. Update last_modified_at.

5) Content Upgrade Layer

Inside major articles add:

  • Free PDF.
  • Execution checklist.
  • Budget template.
  • Weekly follow-up model.

6) Quality Gate before publishing

  • Clear title + meta description.
  • Single H1 only.
  • Hub link + download link + guide link present.
  • No mechanical text repetition.
  • CTAs are helpful, not aggressive.
