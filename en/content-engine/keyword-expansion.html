Keyword Expansion Engine

Goal

Turn search data into an actionable monthly content plan using structured clusters and priority scoring.

A) Core Cluster Matrix

Expense Management

  • Main keyword: إدارة المصاريف
  • Supporting terms: تنظيم المصاريف، ضبط المصروفات، مصاريف شهرية، تقليل المصاريف
  • Intent: Educational + executable

Budgeting

  • Main keyword: الميزانية
  • Supporting terms: الميزانية الشهرية، budget planning، budget template، تقسيم الراتب
  • Intent: Planning + problem solving

Expense Tracking

  • Main keyword: تتبع المصاريف
  • Supporting terms: expense tracking، تطبيق مصاريف، تسجيل المصروفات اليومية
  • Intent: Tool + daily habit

B) Geo Expansion

  • Egypt: المصاريف، مصاريف البيت، المرتب لا يكفي
  • Morocco: الميزانية، budget، تنظيم المصروف اليومي
  • Saudi Arabia: توزيع الراتب، تخفيض المصروف الشهري

C) Search Console Loop (Every 30 Days)

  1. Open Google Search Console.
  2. Export:
    • New queries (last 28 days).
    • Keywords on page 2 (positions 11–20).
    • Low CTR keywords with high impressions.
  3. Select 3 keywords:
    • One for a new article.
    • One to refresh an existing article.
    • One to expand as FAQ content.
  4. Publish or update within the same cycle.

D) CTR Expansion Formula

  • Title = Verb + Core Keyword + Result + Time Frame.
  • Description = Problem + Solution + Practical Outcome.
  • Use numbers when possible (5 steps, 10 minutes, 30 days).

E) Priority Score

  • Direct intent = +3
  • Medium/high volume = +2
  • Easy internal linking = +2
  • Supports conversion = +3

Start each month with the top 5 keywords scoring ≥ 7.
